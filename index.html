Wheel

Wheel is an IDE to build applications for the EV3 and Powered Up. With Wheel you can create text based programs in a simple Pascal/Basic like language.

Wheel is open source. The IDE and compiler are written in JavaScript.

EV3

Compile and run programs EV3 brick. You don't have to install different firmware or use a micro SD card. When you install Wheel you don't have to install the LabView or Scratch applications to compile and upload programs.

Wheel comes with a virtual machine to run your programs on an EV3 brick.

Powered Up

You can control your Powered Up robots, cars and trains with Wheel!

IDE Features

  • Works with standard firmware!
  • Highlight source code editor
  • Code completion
  • Code hints
  • Breakpoints
  • Debugger, variable inspector
  • Image editor
  • Sound editor
  • Direct motor control
  • EV3 File browser
  • Help files

Language Features

  • Text based language
  • Procedural
  • Number, struct, string and pointer types
  • Include files

Simulator Features

  • Direct motor control
  • EV3 Display
  • View motor status
  • View sensor status
  • Simulate sensor input
  • Read button input
  • Connect to brick
  • Connect to Powered Up
  • Supports daisy chaining
