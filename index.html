Wheel

A text based language to run in an EV3 block.

About Wheel

The Wheel compiler and VM are my little side project, the goal of this project is to create a tool which allows you to combine a scripting language with the existing EV3 blocks.

Wheel is a small text based language. The code can be compiled to a .rtf file which can be executed with an EV3 block.

Wheel can be easily connected to existing blocks.

Features:

  • Text based
  • Array, record, pointer types
  • Include files


Getting started

To get started you need the following software installed:

  • nodejs
  • Mindstorms EV3 software

When you've downloaded and installed the application you can compile a file with the following command:

node wheel.js filename.whl

Code

                    #include "lib/screen.whl"

                    proc main()
                        clearScreen()

                        number i

                        for i = 0 to 10
                            number x
                            number y

                            x = i * 20
                            y = i * 12
                            drawNumber(x, y, i)
                        end
                    end
