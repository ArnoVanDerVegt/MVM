Wheel

A text based language to run in an EV3 block.

Text based

Wheel is a small text based language. The code can be compiled to a .rtf file which can be run from an EV3 brick.

Code

                    proc main()
                        number n

                        for n = 10 downto 1
                            printN(n)
                        end
                    end

Standard firmware

You can use wheel addition to the existing blocks. Wheel can be easily connected to existing blocks.
