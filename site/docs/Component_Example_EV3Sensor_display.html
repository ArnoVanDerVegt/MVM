EV3Sensor display

EV3 Sensor display example

This example shows how to use the EV3 sensor display component. The EV3 sensor component can only be used to display status information about sensors, it's not meant to read sensors.

The component can display the following information:

  • A numeric value
  • A color value when the selected type is a color sensor.
