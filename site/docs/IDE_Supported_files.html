Image files

When you open an image file with the .png, .gif, .jpg, .jpeg or .bmp extension then an import wizard will be started to convert the color image to a black and white image.

Icon Extension Description
images/files/rgf.svg rgf Robot graphic file, momochrome image file.
images/files/png.svg png Image file.
images/files/gif.svg gif Image file.
images/files/jpg.svg jpg/jpeg Image file.
images/files/bmp.svg bmp Image file.

Sound files

When you open a sound file with the .mp3, .wav or .ogg extension then an import wizard will be started to convert the sound file to a 8kHz file. Unfortunately the conversion leads to a large loss of sound quality.

Icon Extension Description
images/files/rsf.svg rsf Robot sound file, 8kHz sound sample.
images/files/mp3.svg mp3 Sound file.
images/files/wav.svg wav Sound file.
images/files/ogg.svg ogg Sound file.
