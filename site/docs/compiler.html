Compiler

Precompiler
The compiler consists of several components. First, there's the precompiler. The precompiler reads all files in your project and includes all files into a single file and replaces all de definitions declared with your #define statements.
Compiler
After the precompiler the script compiler compiles the structure of the code to intermediate code which is annotated with type information.

In the final step the compiler uses the relevant code based on the data types and annotations.
