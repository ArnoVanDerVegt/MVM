File format

Rtf
The standard EV3 File block can read data from a .rft file. The Wheel VM read code from a .rtf file with the following structure:

Line Size Expected value Description
1 1 #STRINGS String marker
2 stringCount String value Sring values
.. .. .. ..
3+stringCount 1 #HEAP_SIZE Heap marker
4+stringCount 1 Number value Heap size value
5+stringCount 1 #REG_CODE Code register value marker
6+stringCount 1 Number value Initial code register value
7+stringCount 1 #REG_STACK Stack register value marker
8+stringCount 1 Number value Initial stack register value
9+stringCount 1 #COMMANDS_SIZE Command size value marker
10+stringCount 1 Number value Number of parts
11+stringCount 1 #COMMANDS Command values marker
12+stringCount commandCount Number value Command parts
.. .. .. ..

A single command is stored as 5 numbers:
  • The command
  • The first parameter type
  • The first parameter value
  • The second parameter type
  • The second parameter value
