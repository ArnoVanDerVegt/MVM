Documentation

A text based language to run in an EV3 block.

Syntax

Syntax

Wheel uses a block structure which begins with the declaration of the type or command followed by an expression end closes with and end statement.
                                struct MyStruct
                                    number x, y ; Position
                                end

                                proc main()
                                end
Variables

Variables

The wheel VM only uses one primitive data type: number.

Wheel supports three other data types: string, record, array and pointers. These types are all based on the primitive number type.

A variable is declared by the type followed by the name of the variable.
Loops

Loops

With a for loop you can repeat a block of code a number of times. There are two types of for loops, one counts up from a given start number and the other counts down from the start number.
Conditions

Conditions

With a condition you can execute a block of code if an expression evaluates as true.

The if statement lets you compare two values, with a select you can compare a value to one or more values.

Procedures

Procedures

A procedure allows you to execute a group of commands multiple times.

The declaration of a procedure starts with the proc keyword followed by the name of the procedure followed by a round open and close bracket.
                                proc myProcedure()
                                end
Once you've declared your procedure you can call it like this:
                                proc main()
                                    myProcedure()
                                end
