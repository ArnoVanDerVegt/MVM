Variables

The wheel VM only uses one primitive data type: number.

Wheel supports three other data types: string, struct, array and pointers. These types are all based on the primitive number type.

A variable is declared by the type followed by the name of the variable.

Number type

A number variable can be used for integers and floating point values. 

                    number n

String type

Strings are read only. A string value is a number representing the index in a string list contained in the program file. 

                    string n

Array

An array represents a list. Arrays can be declared by defining the size between brackets after the variable name. The size must be 0 or a positive number. A size of 0 can be used to create pointer types. 

                    number n[10]

Struct

A struct is a representation of a group of variables. At the moment the compiler does not allow struct types in structs. Array fields are allowed. 
                    struct Point
                        number x
                        number y
                    ends

                    Point p

                    proc main()
                        set p.x, 10
                        set p.y, 5
                    endp

Global scope

When a variable is declared outside of a procedure then it's accessible in the entire scope of the program after the declaration. 
                    #project "Global variable demo"

                    number n

                    proc main()
                        set n, 10
                    endp

Local scope

A variable declared inside a procedure can only be accesed inside that procedure. 
                    #project "Local variable demo"

                    proc main()
                        number n
                        set n, 10
                    endp

Parameters

All procedures except the main procedure can have one or more parameters. 
                    #project "Parameter demo"

                    proc first(number n)
                    endp

                    proc second(number n)
                    endp

                    proc main()
                        first(15)
                        second(17)
                    endp
